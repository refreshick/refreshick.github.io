# Сайт веб-студии (3 страницы)

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub с именем `вашлогин.github.io`
2. Загрузите все файлы (index.html, portfolio.html, contact.html, папку css/) в репозиторий
3. В настройках репозитория найдите раздел "Pages"
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`
5. Сохраните изменения. Сайт будет доступен по адресу: `https://вашлогин.github.io`

## Проверка сниппетов

1. **Google Rich Results Test:**
   - Перейдите: https://search.google.com/test/rich-results
   - Введите URL вашей страницы (например, `https://вашлогин.github.io/`)
   - Сделайте скриншот результата проверки

2. **Яндекс.Вебмастер:**
   - Добавьте сайт в Яндекс.Вебмастер
   - Используйте инструмент "Анализ страниц"
   - Проверьте каждую из трёх страниц
   - Сохраните скриншоты

### Анализ конкурента (пример: webcanape.ru)

**Что сделано хорошо у конкурента:**
1. Чёткая структура title: [Услуга] + [Город] + [Цена/Преимущество]
2. Description содержит УТП и призыв к действию
3. Используются семантические теги (header, section, article)
4. Высокая скорость загрузки (90+ по PageSpeed)

**Что можно улучшить на нашем сайте:**
1. Добавить микроразметку Schema.org для услуг и отзывов
2. Увеличить контентную часть на главной странице
3. Добавить блог для привлечения низкочастотных запросов
4. Оптимизировать изображения (сжать без потери качества)

### Выводы о влиянии ключей на видимость

1. **Высокочастотные ключи** ("создание сайтов", "веб-разработка") - дают общий трафик, но высокая конкуренция. Требуют идеальной оптимизации всех элементов страницы.

2. **Среднечастотные ключи** ("сайт под ключ", "корпоративный сайт") - наиболее эффективны для коммерческих сайтов. Конверсия выше, так как пользователь точно знает, что ищет.

3. **Низкочастотные ключи** ("сайт визитка для салона красоты Москва") - легче вывести в топ, дают целевой трафик с высокой конверсией. Рекомендуется создавать отдельные посадочные страницы под такие запросы.

**Рекомендации:**
- Для главной страницы использовать ВЧ+СЧ ключи
- Для страниц услуг - СЧ+НЧ ключи с геопривязкой
- Регулярно обновлять портфолио с описанием технологий (это привлекает НЧ запросы)
- Добавить FAQ-раздел с ответами на частые вопросы (покрывает НЧ запросы)
